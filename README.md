# Prajwal AI

A modern, highly-capable AI assistant for Android, built with Flutter. Prajwal AI acts similarly to Google Assistant and ChatGPT, with a strong focus on privacy, voice support, and personalization. 

## Features
- **Personalized Interaction:** Addresses the owner as "Prajwal" and handles context.
- **Guest Mode:** Secures private information and prompts guests for their names.
- **Voice Support & Multilingual:** Understands English, Kannada, Hindi, Tamil, and Telugu (integration hooks included).
- **Device & App Control:** (Mocked hooks) Navigate applications, camera, and tools.
- **Dark/Light Mode:** Responsive UI following Material 3 guidelines.

## Running the App
```bash
flutter pub get
flutter run
```

## Tech Stack
- **Flutter:** UI Toolkit
- **Provider:** State Management

## Setup
No sensitive API keys are exposed. Simply compile the Flutter project for Android. Ensure permissions for Microphone are granted when prompted.
